# Python TCP Port Scanner

A simple **multi-threaded TCP Port Scanner** built using Python.  
This tool scans a target host across a specified range of ports and identifies whether the ports are **OPEN or CLOSED**.

The project demonstrates fundamental concepts of **networking, socket programming, and concurrency using threads**.

---

##  Features

- Scan a **custom range of TCP ports**
- Detect **open and closed ports**
- Uses **multi-threading** for faster scanning
- Simple command line interface
- Demonstrates **socket programming concepts**

---

##  Technologies Used

- Python
- Socket Programming
- Threading
- Basic Networking Concepts
